# Changelog
All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- Ease Animation curves preset
- Animation Curve Extensions
- Position Animation component
- Scale Animation component
- TextMesh Pro Visible Characters Animation component

## [0.3.0] - 2026-07-10
### Added
- Core animations: AlphaAnimation and RotationAnimation

### Removed
- LitMotion and LitMotion.Animation packages (not compatible with Unity 6.5)

## [0.2.0] - 2026-05-27
### Added
- TMP Text Visible Characters Animation 

### Changed
- Play animation when instance is enabled

## [0.1.0] - 2026-04-04
### Added
- AnimationSequence
- AbstractAnimation
- LitMotion 2.0.1 and LitMotion.Animation 2.0.0 packages
- CHANGELOG
- Package file
- README
- LICENSE
- gitignore
- Initial commit
